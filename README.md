# Qt-Text-Style
#一个简单的 Qt 桌面应用程序，演示了信号与槽机制的基本使用
可实现的功能如下 
- 字体设置：支持 Underline（下划线）、Italic（斜体）、Bold（粗体）
- 复选框切换
- 颜色选择：提供 Black（黑色）、Red（红色）、Blue（蓝色）单选按钮选择
- 文本显示：实时预览字体样式和颜色效果 - 按钮操作：包含退出、确定、清空等基本功能按钮
